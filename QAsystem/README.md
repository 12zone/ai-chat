# 规章制度数据说明

## 项目概述

本项目是一个基于Spring Boot的问答系统，用于管理和检索规章制度信息。系统使用H2内存数据库存储文件信息，使用OpenAI API进行文本嵌入，使用DeepSeek API进行大模型交互。

## 数据源说明

### 原始数据来源
- **文件路径**：`crawled_data/regulations_20260310_230607.json`
- **数据类型**：JSON格式的规章制度数据
- **爬取时间**：2026-03-10
- **数据来源**：西安建筑科技大学教务处网站
- **数据内容**：包含国家法律法规和学校内部规章制度

### 数据处理流程
1. **数据爬取**：从学校教务处网站爬取规章制度信息
2. **数据清洗**：去除HTML标签，提取纯文本内容
3. **数据转换**：将原始数据转换为系统所需的JSON格式
4. **数据导入**：系统启动时自动从`data.json`导入数据到数据库

## 数据文件结构

### 原始数据文件 (`regulations_20260310_230607.json`)
包含以下字段：
- `id`：文档唯一标识
- `title`：文档标题
- `content`：文档内容（纯文本）
- `htmlContent`：文档内容（HTML格式）
- `firstLevelCategory`：一级分类
- `secondLevelCategory`：二级分类
- `url`：原文链接
- `sourceUrl`：来源链接
- `publishDate`：发布日期
- `crawlTime`：爬取时间
- `contentType`：内容类型
- `imageUrls`：图片URL列表
- `localImagePaths`：本地图片路径列表
- `imageOcrTexts`：图片OCR文本列表

### 处理后数据文件 (`data.json`)
包含以下字段：
- `title`：文档标题
- `category`：分类（格式：一级分类/二级分类）
- `date`：发布日期
- `content`：文档内容（纯文本）

## 字段说明

| 字段名 | 类型 | 描述 | 示例值 |
|-------|------|------|--------|
| title | String | 文档标题，包含法规名称和发布日期 | 中华人民共和国教育法 2005-10-03 |
| category | String | 文档分类，采用多级分类结构 | 规章制度/国家法律法规 |
| date | String | 发布日期，格式为YYYY-MM-DD | 2005-10-03 |
| content | String | 文档内容，纯文本格式，包含完整的法规条文 | 国家主席令第45号 1995年3月18日第八届全国人民代表大会第三次会议通过... |

## 数据导入机制

系统启动时，会自动执行以下步骤：
1. 检查`src/main/resources/data.json`文件是否存在
2. 如果存在，读取文件内容并解析为`File`对象
3. 将解析后的对象保存到数据库中
4. 对保存的文件进行向量化处理，存储到向量数据库
